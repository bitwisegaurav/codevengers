Control Structures in VB.Net

Control structures are used to control the flow of execution of the program. In VB.Net, the control structures are categorized into three types:

  • Decision Making Structures/Selection Statements
  • Looping Structures
  • Jump Statements
  • Branching Statements

Decision Making Structures/Selection Statements

Decision making structures are used to execute a block of code based on the result of an expression. The decision making structures in VB.Net are:

  • If Then End If
  • If Then Else End If
  • If Then ElseIf End If
  • Select Case
  • Immediate if (IIF)

If then End if

The If Then End If statement is used to execute a block of code if the condition is true. If the condition is false, the code block will not be executed.

Syntax : 

If condition Then
    'Statements
End If

Example : 

Dim a As Integer = 10
If a > 5 Then
    Console.WriteLine("a is greater than 5")
End If

If Then Else End If

The If Then Else End If statement is used to execute one block of code if a specified condition is true, and another block of code if the condition is false. 

Dim a As Integer = 10
If a > 5 Then
    MessageBox("a is greater than 5")
Else
    MessageBox("a is not greater than 5")
End If

If Then ElseIf End If

The If Then ElseIf End If statement is used to select one of many blocks of code to be executed. 

Dim a As Integer = 10
If a > 10 Then
    MessageBox("a is greater than 10")
ElseIf a = 10 Then
    MessageBox("a is equal to 10")
Else
    MessageBox("a is less than 10")
End If

Select Case

The Select Case statement is used to select one of many blocks of code to be executed. 

Dim a As Integer = 10
Select Case a
    Case 1
        MessageBox("a is 1")
    Case 2
        MessageBox("a is 2")
    Case 10
        MessageBox("a is 10")
    Case Else
        MessageBox("a is not 1, 2, or 10")
End Select

Immediate if (IIF)

The Immediate if (IIF) function returns one of two values depending on the evaluation of an expression. 

Dim a As Integer = 10
Dim result As String = IIf(a > 5, "a is greater than 5", "a is not greater than 5")
MessageBox(result)

Looping Structures

Looping structures are used to execute a block of code multiple times. The looping structures in VB.Net are:

  • For Next
  • For Each Next
  • Do While Loop
  • Do Until Loop
  • While End While

For Next

The For Next loop is used to execute a block of code a specified number of times.

For i As Integer = 1 To 5
    MessageBox("This is loop number " & i)
Next

For Each Next

The For Each Next loop is used to execute a block of code for each element in a collection.

Dim numbers As Integer() = {1, 2, 3, 4, 5}
For Each num In numbers
    MessageBox("The current number is " & num)
Next

Do While Loop

The Do While Loop is used to execute a block of code as long as a specified condition is true.

Dim i As Integer = 1
Do While i <= 5
    MessageBox("This is loop number " & i)
    i += 1
Loop

Do Until Loop

The Do Until Loop is used to execute a block of code until a specified condition is true.

Dim i As Integer = 1
Do Until i > 5
    MessageBox("This is loop number " & i)
    i += 1
Loop

While End While

The While End While loop is used to execute a block of code as long as a specified condition is true.

Dim i As Integer = 1
While i <= 5
    MessageBox("This is loop number " & i)
    i += 1
End While

Jump Statements

Jump statements are used to transfer control to another part of the program. The jump statements in VB.Net are:

  • GoTo
  • Return
  • Stop
  • End

GoTo

The GoTo statement transfers control to the labeled statement in the same procedure.

Dim number As Integer = 1
Start:
If number <= 5 Then
    MessageBox("Number is " & number)
    number += 1
    GoTo Start
End If

Return

The Return statement terminates the execution of the function and returns control to the calling code.

Function Add(a As Integer, b As Integer) As Integer
    Return a + b
End Function

Dim result As Integer = Add(5, 10)
MessageBox("Th result is " & result)

Stop

The Stop statement suspends execution, but unlike End, program execution can be resumed.

For i As Integer = 1 To 10
    If i = 5 Then
        Stop
    End If
    MessageBox("Number is " & i)
Next

End

The End statement stops code execution abruptly, and starts the tear-down process for the .NET runtime.

For i As Integer = 1 To 10
    If i = 5 Then
        End
    End If
    MessageBox("Number is " & i)
Next

Branching Statements

Branching statements are used to transfer control to another part of the program based on the result of an expression. The branching statements in VB.Net are:

  • Exit
  • Continue

Exit

The Exit statement is used to exit a procedure or block of code immediately.

    For i As Integer = 1 To 10
        If i = 5 Then
            Exit For
        End If
        MessageBox("Number is " & i)
    Next

Continue

The Continue statement is used to skip the rest of the current loop iteration and continue with the next iteration.

    For i As Integer = 1 To 10
        If i = 5 Then
            Continue For
        End If
        MessageBox("Number is " & i)
    Next

Conclusion

In this tutorial, we have covered the various control structures in VB.Net, including decision-making structures, looping structures, jump statements, and branching statements. Understanding these structures is fundamental to programming in VB.Net as they allow you to control how your code is executed based on conditions and loops.
