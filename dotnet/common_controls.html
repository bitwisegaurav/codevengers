Commonly used Controls in VB.Net

Controls are the building blocks of a Windows Forms application. The controls are used to create the user interface of the application. The commonly used controls in VB.Net are :

  • Form
  • Label
  • TextBox
  • Button
  • CheckBox
  • RadioButton
  • GroupBox
  • PictureBox
  • Timer
  • ProgressBar
  • ListBox
  • ComboBox
  • DateTimePicker
  • Line and Shape Controls

Form

The Form control is used to create a window that contains other controls. The form is the main window of the application. The form is the container of all the controls.

A Form can be developed using System.Windows.Forms.Form class which is available in System Namespace.

Properties of Form Control

  • BackColor : Sets the background color of the form. Example: 
    Form1.BackColor = Color.Red
  • BackgroundImage : Sets the background image of the form. Example: 
    Form1.BackgroundImage = Image.FromFile("C:\background.jpg")
  • BorderStyle : Sets the border style of the form. Example: 
    Form1.FormBorderStyle = FormBorderStyle.Fixed3D
  • Text : Sets the text of the form's title bar. Example: 
    Form1.Text = "Hello World"
  • Size : Sets the size of the form. Example: 
    Form1.Size = New Size(300, 200)
  • StartPosition : Sets the starting position of the form. Example: 
    Form1.StartPosition = FormStartPosition.CenterScreen
  • WindowState : Sets the window state of the form. Example: 
    Form1.WindowState = FormWindowState.Maximized
  • MaximizeBox : Determines whether the Maximize box is displayed. Example: 
    Form1.MaximizeBox = False
  • MinimizeBox : Determines whether the Minimize box is displayed. Example: 
    Form1.MinimizeBox = False
  • Icon : Sets the icon for the form. Example: 
    Form1.Icon = New Icon("C:\icon.ico")
  • Opacity : Sets the opacity level of the form. Example: 
    Form1.Opacity = 0.5
  • TopMost : Sets whether the form should be displayed on top of all other windows. Example: 
    Form1.TopMost = True
  • TransparencyKey : Sets a color to be transparent in the form. Example: 
    Form1.TransparencyKey = Color.White

Methods of Form Control

  • Show() : Displays the form. Example: 
    Form1.Show()
  • ShowDialog() : Displays the form as a modal dialog box. Example: 
    Form1.ShowDialog()
  • Hide() : Hides the form. Example: 
    Form1.Hide()
  • Close() : Closes the form. Example: 
    Form1.Close()
  • Activate() : Activates the form. Example: 
    Form1.Activate()
  • Deactivate() : Deactivates the form. Example: 
    Form1.Deactivate()
  • Focus() : Sets input focus to the form. Example: 
    Form1.Focus()
  • Refresh() : Forces the form to invalidate its client area and immediately redraw itself. Example: 
    Form1.Refresh()
  • Update() : Causes the form to redraw the invalidated regions within its client area. Example: 
    Form1.Update()
  • Invalidate() : Invalidates a specific region of the form and causes a paint message to be sent to the form. Example: 
    Form1.Invalidate()
  • PerformLayout() : Forces the form to apply layout logic to all its child controls. Example: 
    Form1.PerformLayout()

Events of Form Control

  • Load : Occurs before a form is displayed for the first time. Example: 
    Private Sub Form1_Load(sender As Object, e As EventArgs) Handles MyBase.Load
  • Activated : Occurs when the form is activated in code or by the user. Example: 
    Private Sub Form1_Activated(sender As Object, e As EventArgs) Handles MyBase.Activated
  • Closed : Occurs when the form is closing. Example: 
    Private Sub Form1_Closed(sender As Object, e As EventArgs) Handles MyBase.Closed
  • Closing : Occurs when the form is about to close. Example: 
    Private Sub Form1_Closing(sender As Object, e As CancelEventArgs) Handles MyBase.Closing
  • Deactivate : Occurs when the form loses focus and is no longer the active form. Example: 
    Private Sub Form1_Deactivate(sender As Object, e As EventArgs) Handles MyBase.Deactivate
  • FormClosed : Occurs after the form is closed. Example: 
    Private Sub Form1_FormClosed(sender As Object, e As FormClosedEventArgs) Handles MyBase.FormClosed
  • FormClosing : Occurs as the form is being closed. Example: 
    Private Sub Form1_FormClosing(sender As Object, e As FormClosingEventArgs) Handles MyBase.FormClosing
  • GotFocus : Occurs when the form receives focus. Example: 
    Private Sub Form1_GotFocus(sender As Object, e As EventArgs) Handles MyBase.GotFocus
  • LostFocus : Occurs when the form loses focus. Example: 
    Private Sub Form1_LostFocus(sender As Object, e As EventArgs) Handles MyBase.LostFocus
  • Shown : Occurs whenever the form is first displayed. Example: 
    Private Sub Form1_Shown(sender As Object, e As EventArgs) Handles MyBase.Shown
  • VisibleChanged : Occurs when the Visible property value changes. Example: 
    Private Sub Form1_VisibleChanged(sender As Object, e As EventArgs) Handles MyBase.VisibleChanged

Label

The Label control is used to display text on the form. The label control is used to provide information to the user. The label control is a non-editable text control.

Properties of Label

  • Text : Sets the text of the label. Example: 
    Label1.Text = "Hello World"
  • Font : Sets the font of the label. Example: 
    Label1.Font = New Font("Arial", 12, FontStyle.Bold)
  • ForeColor : Sets the foreground color of the label. Example: 
    Label1.ForeColor = Color.Red
  • BackColor : Sets the background color of the label. Example: 
    Label1.BackColor = Color.Yellow
  • AutoSize : Determines whether the label resizes itself to fit its contents. Example: 
    Label1.AutoSize = True
  • BorderStyle : Sets the border style of the label. Example: 
    Label1.BorderStyle = BorderStyle.FixedSingle
  • TextAlign : Sets the alignment of the text in the label. Example: 
    Label1.TextAlign = ContentAlignment.MiddleCenter
  • Image : Sets the image to be displayed in the label. Example: 
    Label1.Image = Image.FromFile("C:\image.jpg")
  • ImageAlign : Sets the alignment of the image in the label. Example: 
    Label1.ImageAlign = ContentAlignment.MiddleLeft
  • ImageIndex : Sets the index of the image in the image list control. Example: 
    Label1.ImageIndex = 0
  • ImageKey : Sets the key of the image in the image list control. Example: 
    Label1.ImageKey = "image1"
  • ImageList : Sets the image list control that contains the image to be displayed in the label. Example: 
    Label1.ImageList = ImageList1
