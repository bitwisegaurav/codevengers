Fundamentals of VB.NET

Keywords

Keywords are reserved words that have special meaning in the language. They are used to define the syntax and structure of the language.

Reserved Keywords

Reserved Keywords cannot be used as identifiers. However, this restriction can be overlooked by enclosing reserved keywords in square brackets [].

Some of the keywords in VB.NET are:

  • Dim
  • Class
  • Double
  • Call
  • ByVal
  • Continue

Unreseved Keywords

Unreserved Keywords can be used as identifiers but this should be avoided for better programming practices. Some of the unreserved keywords in VB.NET are:

  • Aggregate
  • Custom
  • Into
  • Equals
  • isTrue
  • Text

Identifiers

Identifiers are names given to variables, methods, classes, etc. in a program. Identifiers are case-insensitive in VB.NET.

Rules for Identifiers

  • Identifiers must begin with an alphabet or an underscore.
  • Identifiers can contain alphabets, digits, and underscores.
  • Identifiers cannot contain spaces or special characters except for underscores.
  • Identifiers cannot be reserved keywords.
  • Identifiers are case-insensitive.

Data types

Data types are used to define the type of data that a variable can store.

Numeric Data Types

  • Integer : It stores whole numbers without fractional parts. It can be signed (can store both positive and negative values) or unsigned (store only positive values).
    • Byte : 1 byte
    • Short : 2 bytes
    • Integer : 4 bytes
    • Long : 8 bytes
  • Floating point : It can store numbers with fractional parts. Single (provide 6 digits of accuracy) and Double (provide 14 digits of accuracy) are used for high precision calculation.
    • Single : 4 bytes
    • Double : 8 bytes
  • Decimal : It holds a binary integer value, together with a sign bit and an integer caling factor that is used to specify the portion of the value as a decimal fraction.
    • Decimal : 16 bytes

Non-Numeric Data Types

Char, String, Date, Boolean, User-defined and Object are data types are used in this type.

Char, Boolean, User-defined and Date are of value type while String and Object are of reference type.

  • Char : It stores a single character.
  • String : It stores a sequence of characters.
  • Date : It stores date and time values.
  • Boolean : It stores true or false values.
  • User-defined : It is a custom data type created by the user.
  • Object : It is a reference type that can store any type of data.

Variables

Variables are used to store data in a program. They are used to store values that can be used later in the program.

Declaring a Variable

It is done using the Dim keyword. A declaration statement of a variable describles name, data type and scope of variable.

Syntax : 

Dim variable_name As data_type

Example : 

Dim Age As Integer

Assigning Value

It is done using the = operator. It is used to assign a value to a variable.

Syntax : 

variable_name = value

Example : 

Age = 10

Check Data Type of Variable

It is done using the GetType method. It is used to check the data type of a variable.

Syntax : 

variable_name.GetType()

Example : 

MessageBox(Age.GetType().ToString())

Option Explicit Statement

Option Explicit On statement is used to force the declaration of variables before using them in a program. It is used to avoid the use of undeclared variables.

Syntax : 

Option Explicit On

Option Strict Statement

Option Strict On imples that type checking switch is on.

Literals

Literals are constant values that are used in a program. They are used to represent fixed values in a program.

For exmaple : 

Dim integerLiteral As Integer = 10

Dim floatingPointLiteral As Double = 3.14

Dim stringLiteral As String = "Hello, world!"

Dim charLiteral As Char = "A"c

Dim booleanLiteral As Boolean = True

Dim dateLiteral As Date = #12/31/2021#

Here 10, 3.14, "Hello, world!", "A"c, True and #12/31/2021# are literals where integerLiteral, floatingPointLiteral, stringLiteral, charLiteral, booleanLiteral and dateLiteral are variables

  • Integer Literal : It is used to store integer values.
  • Floating Point Literal : It is used to store floating-point values.
  • String Literal : It is used to store string values.
  • Char Literal : It is used to store character values.
  • Boolean Literal : It is used to store boolean values.
  • Date Literal : It is used to store date values.

Constants or Enumerations

Constants are used to store fixed values that do not change during the execution of a program. Constants are declared using the Const keyword.

User defined Constants

It is done using the Const keyword. It is used to declare a constant.

Syntax : 

Modifier Const constant_name As data_type

Example : 

Const pi As Double = 3.14

Modifier is optional and can be Public, Private, Protected, Friend, Protected Friend, Private Protected or Protected Friend.

User Defined Enumerations

Enumerations are used to create a group of named constants. Enumerations are declared using the Enum keyword.

Syntax : 

Enum enum_name
    enum_value1
    enum_value2
    ...
    enum_valueN
End Enum

Instrinsic constants

Instrinsic constants are system defined constants which are built in the environment and need not to be defined explicitly.

Some expamples are :

  • vbCr : carriage return character.
  • vbLf : linefeed character.
  • vbCrLf : carriage return and line feed character.
  • vbNewLine : newline character.
  • vbNullChar : null character.

Scope of Variable

The scope of a variable determines its visibility or accessibility from different parts of your program. In VB.NET, variables can have different levels of scope:

  • Namespace Level : Variables declared outside any procedure or block but within a namespace. These variables are accessible from any code in the same namespace.
  • Module Level : Variables declared outside any procedure but inside a module, class, or structure. These variables are accessible from any code in the same module, class, or structure.
  • Procedure Level : Variables declared inside a procedure. These variables are only accessible from within the same procedure.
  • Block Level : Variables declared inside a block of code, such as within a loop or conditional statement. These variables are only accessible from within the same block.

Choosing the right scope for your variables is important for maintaining clean, understandable code and avoiding naming conflicts and unexpected behavior.

Operators

An operator is symbol that tells the computer to perform certain manipulations on operands.

Types of operator

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on numeric operands. The following are the arithmetic operators in VB.NET:

  • Addition (+) : Performs the addition of two operands.
  • Subtraction (-) : Performs the subtraction of two operands.
  • Multiplication (*) : Performs the multiplication of two operands.
  • Division (/) : Performs the division of two operands.
  • Exponentiation (^) : Raises the first operand to the power of the second operand.
  • Integer Division (\) : Performs the division of two operands and returns the integer quotient.
  • Modulus (%) : Returns the remainder of the division of two operands.

Example : 

Dim a As Integer = 10
Dim b As Integer = 5
Dim c As Integer = a + b
Dim d As Integer = a - b
Dim e As Integer = a * b
Dim f As Integer = a / b
Dim g As Integer = a ^ b
Dim h As Integer = a \ b
Dim i As Integer = a Mod b

MessageBox("Addition : " & c)
MessageBox("Subtraction : " & d)
MessageBox("Multiplication : " & e)
MessageBox("Division : " & f)
MessageBox("Exponentiation : " & g)
MessageBox("Integer Division : " & h)
MessageBox("Modulus : " & i)

Relational Operators

Relational operators are used to compare two operands. The following are the relational operators in VB.NET:

  • Equal to (=) : Checks if the values of two operands are equal.
  • Not equal to (<>) : Checks if the values of two operands are not equal.
  • Greater than (>) : Checks if the value of the left operand is greater than the value of the right operand.
  • Less than (<) : Checks if the value of the left operand is less than the value of the right operand.
  • Greater than or equal to (>=) : Checks if the value of the left operand is greater than or equal to the value of the right operand.
  • Less than or equal to (<=) : Checks if the value of the left operand is less than or equal to the value of the right operand.

Example : 

Dim a As Integer = 10
Dim b As Integer = 5

MessageBox("Equal to : " & (a = b))
MessageBox("Not equal to : " & (a <> b))
MessageBox("Greater than : " & (a > b))
MessageBox("Less than : " & (a < b))
MessageBox("Greater than or equal to : " & (a >= b))
MessageBox("Less than or equal to : " & (a <= b))

Logical Operators

Logical operators are used to combine two or more conditions. The following are the logical operators in VB.NET:

  • And : Returns True if both conditions are True.
  • Or : Returns True if at least one of the conditions is True.
  • Not : Returns the inverse of a condition.

Example : 

Dim a As Boolean = True
Dim b As Boolean = False

MessageBox("And : " & (a And b))
MessageBox("Or : " & (a Or b))
MessageBox("Not : " & Not a)

Assignment Operators

Assignment operators are used to assign values to variables. The following are the assignment operators in VB.NET:

  • Simple assignment (=) : Assigns the value of the right operand to the left operand.
  • Add and assign (+=) : Adds the right operand to the left operand and assigns the result to the left operand.
  • Subtract and assign (-=) : Subtracts the right operand from the left operand and assigns the result to the left operand.
  • Multiply and assign (*=) : Multiplies the right operand by the left operand and assigns the result to the left operand.
  • Divide and assign (/=) : Divides the left operand by the right operand and assigns the result to the left operand.

Example : 

Dim a As Integer = 10
Dim b As Integer = 5

a += b
MessageBox("Add and assign : " & a)

a -= b
MessageBox("Subtract and assign : " & a)

a *= b
MessageBox("Multiply and assign : " & a)

a /= b
MessageBox("Divide and assign : " & a)

Bitwise Operators

Bitwise operators are used to perform operations on binary representations of numeric operands. The following are the bitwise operators in VB.NET:

  • Bitwise And (&) : Performs a bitwise AND operation on the operands.
  • Bitwise Or (|) : Performs a bitwise OR operation on the operands.
  • Bitwise Not (~) : Performs a bitwise NOT operation on the operand.
  • Bitwise Xor (^) : Performs a bitwise XOR operation on the operands.
  • Bitwise shift left (<<) : Shifts the bits of the left operand to the left by the number of positions specified by the right operand.
  • Bitwise shift right (>>) : Shifts the bits of the left operand to the right by the number of positions specified by the right operand.

Example : 

Dim a As Integer = 10 ' Binary: 1010
Dim b As Integer = 5  ' Binary: 0101

MessageBox("Bitwise And : " & (a And b))
MessageBox("Bitwise Or : " & (a Or b))
MessageBox("Bitwise Not : " & Not a)
MessageBox("Bitwise Xor : " & (a Xor b))
MessageBox("Bitwise shift left : " & (a << 1))
MessageBox("Bitwise shift right : " & (a >> 1))

Concatenating Strings

Concatenation is the process of combining two or more strings into a single string. In VB.NET, you can concatenate strings using the & or + operator or the String.Concat method.

Example : 

Dim firstName As String = "John"
Dim lastName As String = "Doe"

MessageBox(firstName & " " & lastName)
MessageBox(firstName + " " + lastName)
MessageBox(String.Concat(firstName, " ", lastName))
