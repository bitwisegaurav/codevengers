EDI

Electronic Data Interchange (EDI) is the structured exchange of documents between companies by electronic mean without any human intervention.

A Specified format is set by both the parties/companies to facilitate transmission of information.

Some EDI documents are : Invoices, Purchase orders, Financial information letters, Transaction Bills, Claims, etc.

Purpose

  • Automated flow of business information
  • Integrated information flows
  • Common language and data format among different computers

Process

The basic process of EDI is same as manual. The difference is that EDI transactions are sent and received electrnoically as packets of data formatted.

  1. The sender generates an electronic file with necessary information instead of purchase order.
  2. Then this document is translated into an agreed EDI standard format, and this process is called mapping.
  3. This document is then send electrnoically either through a VAN (Value Added Network), direct connection via AS2, a web based EDI tool or outsourcing with an EDI service provider.
  4. Then receiver translates the file to a file that is usable by their Business.
  5. An acknowledgement document is generated and sent to the originating organization.

Components of EDI

EDI Standards

A set of rules, agreed upon, accepted and voluntarily adhered to, by which the data is structured into message formats for exchange of business and operational information.

  • ANSI X12 : A standard for electronic data interchange (EDI) that defines the format and structure of business documents exchanged between different computer systems.
  • EDIFACT : An international standard for electronic data interchange (EDI) that provides a set of rules for structuring, formatting, and exchanging business documents between organizations.

EDI Software

The major functions of this are data conversion, formatting and message communication.

  • Application Software : Software used for specific business functions within the EDI system.
  • Interface Software : Software that enables communication between different systems or components.
  • Translation Software : Software that converts data from one format to another, ensuring compatibility between different systems.
  • Communication Software : Software that facilitates the transmission of EDI messages between trading partners.
