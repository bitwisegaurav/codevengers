Recursion and Recurrence Relations

Sequence

A sequence is a set of numbers that are related to each other by a rule.

For example 1, 2, 3, 4, ... etc. Here an = n.

Recursion

Recursion is a method where the solution to a problem depends on solutions to smaller instances of the same problem.

or

A technique of defining a function, a set or an algorithm in terms of itself is a recursion.

For example 0, 1, 1, 2, 3, 5, 8, ... etc. Here an+1 = an + 1.

Recurrence Relation

A recurrence relation is an equation that recursively defines a sequence where the next term is a function of the previous terms.

or

A relation which is recursive in nature is known as recurrence relation.

A recurrence relation is also called a difference equation.

Important Terms

  • Base Case : The first term of a sequence.
  • Boundary Conditions : Given values of functions.
  • Induction Step : The step by which a sequence is built.
  • Order : The order of a recurrence relation is the difference between highest and lowest subscript.
  • Characteristic Equation : It is an equation whose roots are the values of n for which the recurrence relation holds.

    For example: an = 2an-1 + 1, the characteristic equation is r2 - 2r - 1 = 0.

Types of Recurrence Relations

  • First Order Recurrence Relation

    A recurrence relation of the form: an = can-1 + f(n) for n>=1 where c is a constant and f(n) is a known function is called linear recurrence relation of first order with constant coefficient. If f(n) = 0, the relation is homogeneous otherwise non-homogeneous. For example: x n = 2x n-1 – 1, a n = na n-1 + 1, etc.

  • Second Order Linear Homogeneous Recurrence Relation

    A recurrence relation of the form cnan + cn-1an-1 + cn-2an-2 = 0 for n>=2 where c n, c n-1 and c n-2 are real constants with c n != 0 is called a second order linear homogeneous recurrence relation with constant coefficients.

Solving Recurrence Relations

There are different methods to solve recurrence relations, including generating functions. The solutions depend on the roots of the characteristic equation associated with the relation:

  • If the roots are real and distinct, we take an = A (k1)n + B (k2)n as the general solution.
  • If the roots are real and equal, we take an = (A + Bn)kn as the general solution.
  • If the roots are complex, we take an = rn(Acosnθ + Bsinnθ) as the general solution.
